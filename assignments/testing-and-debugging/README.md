# 📘 Assignment: Testing and Debugging in Python

## 🎯 Objective

Learn how to verify Python code with tests and use debugging strategies to find and fix problems in a program.

## 📝 Tasks

### 🛠️ Write Tests for Python Functions

#### Description

Implement the required functions and add tests that confirm each function returns the correct results.

#### Requirements

Completed program should:

- Define `calculate_average(numbers)` that returns the average value for a list of numbers and returns `0` for an empty list.
- Define `is_palindrome(text)` that returns `True` when the text reads the same forwards and backwards, ignoring spaces and case.
- Include at least four assertions or tests that verify both normal and edge-case behavior.

### 🛠️ Debug a Broken Python Program

#### Description

Fix the provided buggy functions so the program runs correctly and prints the expected results.

#### Requirements

Completed program should:

- Correct the buggy `buggy_divide(a, b)` function so it returns `a / b`.
- Correct the buggy `buggy_reverse_words(sentence)` function so it returns the input sentence with words reversed.
- Run the program and confirm the printed output matches the expected results.
